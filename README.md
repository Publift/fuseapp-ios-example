# Fuse App SDK iOS Example

Fuse is the [Publift](https://www.publift.com/) advertising solution for publishers of all sizes across web sites and apps.

This repository contains example code for the Fuse App SDK on iOS.

## Getting started

1. Clone this repo
2. Run a `pod install`
3. Open `FuseAppExample.xcworkspace` in Xcode
4. See the following files for implementation examples:
  - [SwiftUI Examples](blob/main/FuseAppExample/FuseAppExample/BannerView.swift)
  - [UIKit Examples](blob/main/FuseAppExample/FuseAppExample/BannerViewController.swift)
